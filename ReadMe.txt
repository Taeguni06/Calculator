Step2. 계산 기능, 최근 기록 삭제, 마지막 기록 삭제, 전체 조회 기능

Step3. 계산(Create), 조회(Read), 수정(Update), 삭제(Delete) CRUD를 충족하게 작성
       계산. 실수형 계산기, 정수형 계산기로 구분 (데이터 또한 따로 저장됨)
       조회. 전체 조회, 입력한 값보다 큰 값 조회, 입력한 값보다 작은 값 조회
       수정. 리스트에서 인덱스를 입력 후 요소를 수정 가능
       삭제. 전체 삭제, 선택 삭제

       사용법.
       1. 실행 후 사용한 계산기 선택 (1, 2) 또는 종료 (0)

       2. 계산기의 기능 선택
        계산, 조회(history), 수정(fix), 삭제(remove) 입력

        계산. 숫자1, 연산자, 숫자2 입력 후 계산 결과 출력

        조회. 입력시 현재 전체 기록 조회
                1 입력시 큰 값 찾기, 입력한 값보다 큰 값을 리스트에서 출력
                2 입력시 작은 값 찾기, 입력한 값보다 작은 값을 리스트에서 출력
        수정. 현재
